# TMDT - Website Thương Mại Điện Tử

## Mô tả
Website thuê xe gồm frontend và backend.

## Công nghệ
- Frontend: ReactJS / Vue / JS
- Backend: Python (Flask/Django)
- Database: ...

## Cách chạy
### Backend
cd backend
pip install -r requirements.txt
python app.py

### Frontend
cd frontend
npm install
npm start
